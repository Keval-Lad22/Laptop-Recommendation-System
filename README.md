# Laptop Recommendation System

This project uses **Machine Learning** to recommend the best laptops to users based on their preferences and system specifications.

The model analyzes various laptop features such as:
- Brand
- Processor Type
- RAM Size
- Storage Type and Capacity
- Display Size
- Price Range

and suggests the most suitable laptops using trained ML algorithms.

---

# Features

- Recommends laptops based on user preferences  
- Accepts **live user input** through the console  
- Predicts or suggests laptops similar to the user’s requirement  
- Displays **evaluation metrics** such as:
  - RMSE (Root Mean Squared Error)
  - R² Score
